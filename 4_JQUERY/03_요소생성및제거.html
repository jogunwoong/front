요소 생성 및 제거

* 삽입 관련 메소드

선택된 요소를 기준을 새로운 요소를 추가시켜주는 메소드 

        $(A).append(B) : A 요소 내에 뒷부분에 B 요소를 추가 (자손/하위)
        $(A).prepend(B) : A 요소 내에 앞부분에 B 요소를 추가 (자손/하위)

        $(A).after(B) : A 요소 뒷부분에 B 요소를 추가 (동위)
        $(A).before(B) : A 요소 앞부분에 B 요소를 추가 (동위)

* 제거 관련 메소드

  • $ (선택자).empty() : 선택된 요소의 하위 요소를 제거해주는 메소드
  • $ (선택자).remove()|detach() : 선택된 요소를 제거한 후 해당 요소를 반환
    • .remove() : 제거된 요소의 이벤트를 가져오지 않음
    • .detach() : 제거된 요소의 이벤트까지 가져옴
